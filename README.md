# ECON30

Course project repository for ECON30.

## Repository structure
- **data/**: Raw and/or cleaned datasets used in the analysis (not all sources may be included if they are too large or restricted).
- **code/**: Scripts and notebooks for cleaning, analysis, and figures/tables.
- **output/**: Generated results (tables, figures, exported datasets).
- **docs/**: Notes, writeups, and references (if present).

## How to use
1. Put datasets in `data/` (or update paths in the scripts to point to your local data).
2. Run the scripts/notebooks in `code/` to reproduce the analysis.
3. Check `output/` for generated figures/tables.

## Notes
- If you don’t see some of the folders above yet, create them as you add files.
- If any data is private, keep only non-sensitive samples or document how to obtain it.
